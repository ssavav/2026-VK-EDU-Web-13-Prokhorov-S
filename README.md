# Проект AskBMSTU

## 1. Локальный запуск проекта

1. Перейдите в папку проекта.
2. Создайте виртуальное окружение командой 
   `python -m venv venv`
3. Активируйте виртуальное окружение командой
   - Windows: `venv\Scripts\activate`  
   - macOS/Linux: `source venv/Scripts/activate`
4. Установите зависимости командой
   `pip install -r requirements.txt`
5. Примените миграции базы данных
   `python manage.py migrate`
6. Запустите сервер разработки
   `python manage.py runserver`
7. Проект будет доступен по адресу: http://127.0.0.1:8000/

Для остановки работы локального сервера с проектом нажмите сочетание клавиш `Ctrl+C`

## 2. Запуск через docker

Для запуска проекта через Docker, необходимо, чтобы на компьютере был установлен docker

Для запуска:

1. Перейдите в папку проекта.
2. Соберите и запустите контейнеры командой: 
   `docker compose up --build`
3. Проект будет доступен по адресу http://127.0.0.1:8000/

Для остановки контнейнеров выполните команду
`docker compose down`

## 3. Наполнение базы данных тестовыми данными
Для наполнения тестовыми данными запустить команду
`python manage.py fill_db [ratio]`
где ratio - коэффициент наполнения:
пользователей — равное ratio;
вопросов — ratio * 10;
ответы — ratio * 100;
тэгов - ratio;
оценок пользователей - ratio * 200;
